# 발렛파킹 자동 예약 시스템 사용설명서

## 📁 바로가기 파일 (.bat)

| 파일 | 기능 | 언제 사용 |
|------|------|-----------|
| `서버접속.bat` | 서버 SSH 접속 | 서버에 직접 들어가서 작업할 때 |
| `서버상태확인.bat` | 서비스 상태 확인 | 잘 돌아가고 있는지 체크할 때 |
| `서버재시작.bat` | 서비스 재시작 | 문제가 생겼을 때 |
| `코드업데이트.bat` | 코드 업데이트 + 재시작 | GitHub에 코드 수정 후 서버 반영할 때 |

> 더블클릭하면 자동 실행됩니다.

---

## 📱 텔레그램 봇 명령어

| 명령어 | 기능 |
|--------|------|
| `help` | 사용법 + 명령어 목록 |
| `조회` | 현재 예약 리스트 확인 |
| `확인` | 취소 최종 실행 |
| `취소` | 대기 중인 취소 작업 중단 |

### 예약 취소 방법
1. 텔레그램에 유지할 날짜 전송:
```
2026.05
13
14
20
21
```
2. 봇이 유지/취소 목록 보여줌
3. `확인` 입력 → 취소 실행

## ⚙️ 수동 명령어 (서버 접속 후)

```bash
# 상태 확인
sudo systemctl status balet-scheduler balet-bot

# 재시작
sudo systemctl restart balet-scheduler balet-bot

# 중지
sudo systemctl stop balet-scheduler balet-bot

# 로그 확인
sudo journalctl -u balet-scheduler -n 50 --no-pager
sudo journalctl -u balet-bot -n 50 --no-pager
```
